# DjangoPollingApp
## With the help of Python, Django,Django Sessions and DRF we create the Django Poll Questioning app, where user can register and login to see the question to vote and to see viting percentage of each option.
## User can also see their user detail in UserProfile. 
## Only registered Users can view the questions and can vote. 
## Customizations in Django-Admin Panel. 
## And Also Perform the CRUD operations using Django as well as DRF.
